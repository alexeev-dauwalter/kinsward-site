# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Project

Documentation site for Kinsward (a Dwarf Fortress-inspired colony sim). Built with Astro Starlight. Deployed to GitHub Pages at kinsward.dwltr.life. The game source code is in a separate private repository.

## Commands

```bash
pnpm dev        # dev server at localhost:4321
pnpm build      # production build to dist/
pnpm preview    # preview production build
```

Package manager is pnpm. Do not use npm or yarn.

## Content Conventions

All docs live in `src/content/docs/`. Use `.md` for static pages. Use `.mdx` when the page needs JSX components — primarily `<Kbd>` for keyboard shortcuts:

```mdx
import { Kbd } from 'starlight-kbd/components'

Press <Kbd linux="W" /> to move north.
```

The `linux` prop is the default; `mac` and `windows` fall back to it if omitted.

Starlight components (`Card`, `CardGrid`) are available from `@astrojs/starlight/components`.

Use Starlight admonitions for callouts: `:::tip`, `:::note`, `:::caution[Under Development]`.

## Styling

Theme is in `src/styles/custom.css`. Overrides Starlight CSS variables (accent colors, grays, fonts). Tailwind CSS v4 is available via `@tailwindcss/vite` + `@astrojs/starlight-tailwind`.

## Sidebar

Sidebar is explicitly defined in `astro.config.mjs` (not autogenerated). When adding a new page, add a corresponding sidebar entry.

## Deployment

Pushes to `main` trigger `.github/workflows/deploy.yml` which builds and deploys to GitHub Pages. CNAME for kinsward.dwltr.life is written during build.
