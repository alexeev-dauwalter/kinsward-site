Quick reference for all registered keybindings. Mode-specific pages cover detailed interactions (z-level targeting, box selection, command queuing).

## Global

| Action | Key |
|--------|-----|
| Menu / Back | <Kbd linux="Escape" /> |
| Inventory | <Kbd linux="Tab" /> |
| Map | <Kbd linux="M" /> |
| Quick Save | <Kbd linux="F5" /> |
| Quick Load | <Kbd linux="F9" /> |

## Adventure Mode

| Action | Key |
|--------|-----|
| Move | <Kbd linux="W" /> <Kbd linux="A" /> <Kbd linux="S" /> <Kbd linux="D" /> |
| Mine | Left click |
| Place | Right click |
| Companions | <Kbd linux="N" /> |
| Craft | <Kbd linux="C" /> |
| Overseer | <Kbd linux="B" /> |

Hold <Kbd linux="B" /> for 0.3s to quick-peek into Overseer mode; release to return. See [Adventure Mode](/gameplay/adventure-mode/) for z-level targeting, stairs, and surface transitions.

## Overseer Mode

| Action | Key |
|--------|-----|
| Select | Left click |
| Command | Right click |
| Pan | <Kbd linux="W" /> <Kbd linux="A" /> <Kbd linux="S" /> <Kbd linux="D" /> |
| Z-level | Scroll wheel |
| Units | <Kbd linux="N" /> |
| Adventure | <Kbd linux="B" /> |

See [Overseer Mode](/gameplay/overseer-mode/) for tool modes, unit selection, command queuing, and zone operations.

## Inventory

| Action | Key |
|--------|-----|
| Drop | <Kbd linux="Q" /> |
| Move stack | <Kbd linux="Shift" /> + click |

## Map View

| Action | Key |
|--------|-----|
| Next layer | <Kbd linux="Tab" /> |
| Zoom | Scroll wheel |
| Pan | <Kbd linux="W" /> <Kbd linux="A" /> <Kbd linux="S" /> <Kbd linux="D" /> |

## Debug

| Action | Key |
|--------|-----|
| Admin Menu | <Kbd linux="`" /> |
| Debug Picking | <Kbd linux="F3" /> |
| Hover Debug | <Kbd linux="F11" /> |

:::note
Debug controls are only available when the game is launched with `--debug`. See [Installation](/getting-started/installation/) for details.
:::
